# 🍪 New Recipe — Fill-In Form

Copy everything below the line, fill in the blanks, and send it back. You don't need
to know any code — just replace the prompts in `< >`. A developer will add it to the
app, and after the next deploy it shows up on **every** device automatically.

Tips for filling it out:
- **Ingredients:** one per line as `amount | unit | name | note`. Leave the unit blank
  for things counted as whole items (like eggs). The `note` is optional.
  Examples: `2.25 | cups | all-purpose flour |`  ·  `2 | | eggs | large`
- **Steps:** one per line. If a step needs a timer, add `(timer: 11 min)` at the end.
- **Yield** is how many cookies one normal batch makes (used for batch scaling).
